# PF Macros Autoclickers
 Autoclickers that help you spam guns in Phantom Forces
 For all the scripts to work the best, the guns should be in burst mode.

# Compatible weapons
 RPK12 script and app can be also used for AK12 and AK12BR (all in burst mode).
 M16A4 script and app can also be used for MM20 in Entry Point (Roblox).
 M16A4 can be used with any gun that has a 900 RPM.
 RPK12 can be used with any gun that has a 1000 RPM.
 M4 can be used with any gun that has a 950 RPM.

# Controls
 Press `P` to toggle the script on and off (note that you cannot use the letter 'P' key when the script or app is running). Press `Esc` to exit the app or script completely. Left click to activate the macro.
